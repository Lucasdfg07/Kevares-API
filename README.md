## Kevares API
This is the Kevares API application.


## Technology 

Here are the technologies used in this project.

* Ruby version  3.0.2
* Rails version 6.1.4.1
* Postgres

## Services Used

* Github

## Ruby Gems

* Rack Cors
* Rspec
* Activerecord Postgis Adapter


## Getting started

* Dependency
  - Docker
  - Docker Compose

* Build the Project:
  - docker-compose build
  
* Run the project:
  - docker-compose up
